**Length:** 10 minutes

**Script:**

**[0:00-0:30] Introduction**

5. Repeat

```

**[0:30-2:00] Setup**
```
[Screen: Terminal/commands]

"Step 1: Install dependencies
```bash
npm install clawcredit-sdk ethers openai
```

Step 2: Set up environment variables
```bash
export AGENT_WALLET_KEY=your_private_key
export OPENAI_API_KEY=your_openai_key
export CLAWCREDIT_POOL=0x750e...
```

Step 3: Initialize the agent"

[Screen: Code]
```javascript
import { ClawCredit } from 'clawcredit-sdk';
import { ethers } from 'ethers';
import OpenAI from 'openai';

class SelfFundingAgent {
  constructor() {
    this.credit = new ClawCredit({
      privateKey: process.env.AGENT_WALLET_KEY,
      network: 'base-mainnet'
    });
    
    this.openai = new OpenAI({
      apiKey: process.env.OPENAI_API_KEY
    });
  }
}
```
```

**[2:00-4:00] Check Balance & Borrow**
```
[Screen: Code]

"Step 4: Check balance and borrow if needed
```javascript
async checkAndFund() {
  const balance = await this.credit.getBalance();
  console.log(`Current balance: $${balance}`);
  
  if (balance < 20) {
    console.log('Balance low, requesting loan...');
    
    const loan = await this.credit.requestLoan({
      amount: 50,
      duration: 14 // days
    });
    
    console.log(`Loan approved: $${loan.amount}`);
    console.log(`APR: ${loan.apr}%`);
    console.log(`Total due: $${loan.totalDue}`);
  }
}
```

This automatically borrows $50 when balance drops below $20."
```

**[4:00-7:00] Trading Strategy**
```
[Screen: Code]

"Step 5: Execute trading strategy
```javascript
async executeStrategy() {
  // Get market data
  const marketData = await this.getMarketData();
  
  // Ask GPT-4 for analysis
  const analysis = await this.openai.chat.completions.create({
    model: 'gpt-4',
    messages: [{
      role: 'system',
      content: 'You are a crypto trading analyst.'
    }, {
      role: 'user',
      content: `Analyze this market data: ${JSON.stringify(marketData)}`
    }]
  });
  
  // Parse trading signal
  const signal = analysis.choices[0].message.content;
  
  // Execute trade
  if (signal.includes('BUY')) {
    const profit = await this.executeBuy();
    return profit;
  }
  
  return 0;
}
```

Simple strategy: Use GPT-4 to analyze market, execute trades."
```

**[7:00-8:30] Auto-Repayment**
```
[Screen: Code]

"Step 6: Auto-repayment (already handled)
```javascript
async run() {
  // Check and fund
  await this.checkAndFund();

  // x402 automatically deducts 10% of profit
  // No code needed!

  console.log('Repayment handled automatically');
}
```

The x402 hook monitors earnings and auto-repays."

main();

The agent runs 24/7, funding itself as needed."
