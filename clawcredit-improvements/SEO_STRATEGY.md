## SEO Optimization Checklist

### On-Page SEO
- [ ] Title tag: 50-60 characters, keyword first
- [ ] Meta description: 150-160 characters, CTA included
- [ ] H1: Main keyword, single per page
- [ ] H2s: Long-tail keywords, logical structure
- [ ] URL: Short, keyword-rich, hyphen-separated
- [ ] Images: Descriptive filenames, alt text
- [ ] Internal links: 3-5 per post, relevant anchor text
- [ ] External links: 2-3 to authoritative sources
- [ ] Schema markup: Article, Organization, FAQ

### Technical SEO
- [ ] Page speed: <3 seconds
- [ ] Mobile responsive
- [ ] SSL certificate
- [ ] XML sitemap
- [ ] robots.txt configured
- [ ] Canonical URLs
- [ ] 404 page optimized
- [ ] Structured data

### Content Quality
- [ ] 2,000+ words for pillar pages
- [ ] 1,000+ words for regular posts
- [ ] Original research/data
- [ ] Expert quotes
- [ ] Visual elements (charts, images)
- [ ] Video embeds
- [ ] Downloadable resources
- [ ] FAQ section
